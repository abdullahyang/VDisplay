# VDisplay
基于视觉智能的多人场景视频监控平台项目演示

## 注意事项
目标文件夹下需要有命名以top结尾的文件夹、以hor1结尾的文件夹、hor2结尾的文件夹（可选）、hor3结尾的文件夹（可选），每个文件夹内有对应位置数据的txt文件。
